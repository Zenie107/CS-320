Git Interactive Masterpiece

Complete Workflow, Branching, and Semantic Versioning Guide

1. Setup & Init

Copy Starting Fresh git init

Turn a local folder into a Git repository.

Copy Downloading Work git clone [url]

Copy a remote repository to your computer.

2. Daily Workflow

Copy Staging git add .

Stage all current changes for the next commit.

Copy Saving git commit -m "msg"

Save your staged snapshot to history.

Copy Review git status

See what files are modified or staged.

3. Branching & Merging

Copy Isolation git checkout -b [name]

Create a new branch and switch to it immediately.

Copy Integration git merge [branch]

Merge a feature branch into your current branch.

4. Remote Sync

Copy Uploading git push origin [branch]

Send local commits to the online repository.

Copy Updating git pull

Fetch and download updates from your team.

5. Tagging & Versions

Copy Releasing git tag -a v1.0.0 -m "Release"

Create an annotated tag for a specific version.

Copy Syncing Tags git push origin --tags

Upload all your tags to the server.
